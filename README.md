# To-Do List App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This project helps users manage their daily tasks efficiently with a clean and responsive user interface.

---

## Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Data persistence using localStorage (tasks remain after refresh)  
- Responsive and user-friendly design  

---

## Tech Stack

- HTML5  
- CSS3 (Flexbox, custom styling)  
- JavaScript (DOM manipulation, event handling, localStorage)
